***User Stories***

US-1
As a **player**, I want to enter my nickname before starting the game, so that my score can be saved to the leaderboard.

US-2
As a **player**, I want to choose a difficulty level (Easy, Medium, Hard), so that the game matches my skill level.

US-3
As a **player**, I want to see one email at a time, so that I can focus on analysing it carefully.

US-4
As a **player**, I want to mark emails as “Real” or “Fake”, so that I can test my phishing detection skills.

US-5
As a player, I want to see my score and remaining lives, so that I understand my performance during the session.

US-6
As a **player**, I want to see my final result after finishing the game, so that I know how well I performed.

US-7
As a **player**, I want my score to be saved automatically, so that I can appear on the leaderboard.

US-8
As an **administrator**, I want to manage and update email scenarios, so that the game content can evolve and remain relevant.
